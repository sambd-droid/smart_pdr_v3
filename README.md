# 🌱 Potential Denitrification Rate ( Bangladesh)

An interactive web-based application for estimating and visualizing **Potential Denitrification Rate (PDR)** using geospatial and environmental data.

---

## 📌 Overview

The **PDR Web App** is designed to analyze and map denitrification potential in soils, helping researchers and environmental planners understand nitrogen loss in ecosystems.

This application leverages **remote sensing data, raster analysis, and geospatial tools** to provide real-time insights through an intuitive interface.

---

## 🎯 Objectives

- Estimate Potential Denitrification Rate (PDR)
- Visualize spatial distribution of PDR
- Support environmental and agricultural decision-making
- Provide an easy-to-use analytical platform

---

## 🚀 Features

- 🌍 Interactive map-based visualization (Leaflet)
- 📊 Raster data processing and analysis
- 📍 Region selection and spatial querying
- ⚠️ Validation alerts for out-of-bound selections
- 🎛️ User-friendly Shiny dashboard
- 📈 Real-time data interaction

---

## 🛠️ Tech Stack

- **Language:** R  
- **Framework:** Shiny  
- **Geospatial Libraries:** `terra`, `sf`, `leaflet`, `leaflet.extras`  
- **UI Design:** `bslib`  
- **Deployment:** Posit Connect / Shiny Server  

---


## 🌐 Usage
- Launch the application
- Select a region on the interactive map
- Choose dataset (if applicable)
- Analyze PDR values
- View results and spatial outputs
## 📊 Data Sources
- Satellite raster data (e.g., NDVI, soil properties)
- Environmental datasets (temperature, moisture, etc.)
- User-defined spatial inputs
## 🧪 Methodology

The PDR estimation is based on environmental variables and spatial modeling techniques:

- Raster data extraction
- Spatial overlay analysis
- Empirical or model-based calculation of denitrification rates
## ⚠️ Error Handling
Alerts when selected region is outside raster extent
- Input validation for dataset selection
- User-friendly warning dialogs
